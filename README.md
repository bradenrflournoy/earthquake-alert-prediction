# earthquake-alert-prediction
Machine learning project aiming to classify earthquake alerts based on several seismic metrics

Public ML project for machine learning course.  
**Notebook:** `notebooks/analysis.ipynb` - **Sample data:** `data/sample/dataset_sample.csv`

## Data
Source: [Earthquake Alert Prediction Dataset](https://www.kaggle.com/datasets/ahmeduzaki/earthquake-alert-prediction-dataset?select=earthquake_alert_balanced_dataset.csv).  
This repo does **not** include the full dataset. Download it from the link and place the file(s) in `data/raw/`.
